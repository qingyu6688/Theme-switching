# 高级日夜模式切换组件 (Advanced Theme Toggle Slider)

一个精致、流畅且极具高级感的 Web 主题切换组件。支持在“明亮模式（蓝天白云）”与“深色模式（璀璨星空）”之间无缝切换。

## ✨ 特性

- **极致的视觉体验**：
  - 拒绝平庸，采用精选的高级配色方案（青色 + 暖黄 + 深蓝灰），**完全不含紫色调**。
  - 拟物化设计：滑块内部集成**旋转的太阳与月亮**图标。
  - 动态背景：切换时**白云沉降、星星升起**，细节满满。

- **丝滑的交互动画**：
  - 使用 CSS `cubic-bezier` 贝塞尔曲线打造富有弹性的开关手感。
  - 所有元素（背景色、图标、装饰物）均具备流畅的过渡效果。

- **智能与持久化**：
  - **自动检测**：首次访问自动跟随系统的颜色偏好（System Preference）。
  - **状态记忆**：使用 `localStorage` 记住用户的选择，刷新页面不丢失。

## 🛠️ 技术栈

- **HTML5**: 语义化标签与 SVG 图标。
- **CSS3**: CSS 变量 (Custom Properties)、Flexbox 布局、过渡与动画。
- **JavaScript (Vanilla)**: 原生 JS 实现，无任何第三方依赖。

## 🚀 快速开始

1.  下载本项目文件。
2.  在浏览器中直接打开 `index.html` 即可预览效果。
3.  **集成到你的项目**：
    -   **CSS**: 引入 `style.css` 或将其内容复制到你的主样式表中。
    -   **HTML**: 将 `index.html` 中 `.theme-switch-wrapper` 包裹的代码块复制到你需要显示开关的位置。
    -   **JS**: 引入 `script.js`，确保在 DOM 加载完成后执行。
